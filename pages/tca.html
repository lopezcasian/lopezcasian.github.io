Proyecto final del curso de teoría de control avanzado, el proyecto consiste en controlar dos focos por medio de internet remotamente con la finalidad de simular un salón de clases. Se utilizó Raspberry Pi como server, se programó el back-end con Python con framework Flask, en el front-end HTML, CSS, Bootstrap y jQuery, para conectarnos remotamente utilizamos el servicio remot3.it.
